# Benchmark Results

This folder is where `compare.py` saves generated charts
(`comparison_chart.png`) when `matplotlib` is installed.

It is intentionally kept empty in version control (aside from this
file) -- results are meant to be regenerated on your own machine, since
absolute numbers depend heavily on your CPU, OS scheduler, and Python
build. See `docs/07_benchmark.md` for how to interpret relative
differences instead of chasing absolute numbers.
